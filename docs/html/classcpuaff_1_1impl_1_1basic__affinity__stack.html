Detailed Description

template<typename TRAITS>
class cpuaff::impl::basic_affinity_stack< TRAITS >

basic_affinity_stack is used to keep track of affinities as you get and set them. It is basically a wrapper for an affinity_manager that gives you the ability to keep track of what affinities have been and reset them later.
