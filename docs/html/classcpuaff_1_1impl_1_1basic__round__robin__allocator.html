Detailed Description

template<typename TRAITS>
class cpuaff::impl::basic_round_robin_allocator< TRAITS >

basic_round_robin_allocator is a utility class that takes a set of cpus and returns them as requested in a round-robin fashion. It organizes the cpus such that it returns consecutive cpus from different cores if it can.
