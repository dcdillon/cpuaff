Detailed Description

template<typename TRAITS>
class cpuaff::impl::basic_affinity_manager< TRAITS >

basic_affinity_manager is a collection of all the valid cpus. It provides interfaces to get and set cpu affinity as well as ways to classify cpus on the system so that intelligent choices can be made about what affinity a thread should have.
